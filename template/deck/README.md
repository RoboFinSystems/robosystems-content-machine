# deck/

The **Claude Design** stage (Stage 2) lives here.

- `{TICKER}_deck_brief.md` — generated by `just deck-brief {TICKER}`. The hand-off you paste
  into claude.ai/design.
- `{TICKER}_deck.pptx` — the **canonical** 16:9 deck export from Claude Design. `just slice
  {TICKER}` (and the pipeline) converts it to `{TICKER}_deck.pdf` via PowerPoint's native
  renderer (byte-identical to a manual "Best for printing" export), then rasterizes each slide
  into `../charts/png/`. A pre-exported `{TICKER}_deck.pdf` also works — auto-conversion only
  fires when a PPTX is present.

Thumbnails are made in ChatGPT and dropped into `../assets/` (`yt.png` 16:9, `x.png` 5:2,
`spot.png` 1:1); `just slice` ingests them into `../charts/png/`. See `../DESIGN_INSTRUCTIONS.md`.
